Ecuaciones
A = πD² / 4
V = Q / A
Nr = (ρVD) / μ
f = 0.25 / [log10((ε/3.7D) + (5.74/Nr^0.9))]²
HLfricción = f (L/D) (V² / 2g)
K = (Le/D) ft
hLmenor = K (V² / 2g)
HA = Δz + HLfricción + hLmenor,total
Phid = ρgHAQ
Preq = Phid / η
Para flujo laminar: f = 64/Nr. En pérdidas menores, para acero comercial se usa ft según tabla 10.5; en otros materiales se usa el factor de fricción calculado arriba.
